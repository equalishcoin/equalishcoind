Intro
-----
EqualishCoin is a free open source project derived from Bitcoin, with
the goal of providing a long-term energy-efficient crypto-currency.
Built on the foundation of Bitcoin, innovations such as proof-of-stake
help further advance the field of crypto-currency.


Setup
-----
Unpack the files into a directory and run equalishcoin-qt.exe.

EqualishCoin Core is the original EqualishCoin client and it builds the backbone of the network.
However, it downloads and stores the entire history of EqualishCoin transactions;
depending on the speed of your computer and network connection, the synchronization
process can take anywhere from a few hours to a day or more.

See the EqualishCoin documentation at:
  https://equalishcoin.org/
for more help and information.
